.fun

Defines a pattern matching function. Exhaustiveness and redundancies are checked.

Parameters

  • o : object → The pattern matching map (pattern, function)

Return
Returns a function that when called with an appropriate argument matches the pattern.

Usage

var sum = $p.fun({
    '[]'    : function () { return 0 },
    'x::xs' : function (x, xs) { return x + sum(xs) }
});
